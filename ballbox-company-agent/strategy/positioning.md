# Positioning

## Current thesis

Ballbox places a physical unit in padel centers that monetizes the same footprint twice:

- product sales: ball tubes
- advertising sales: on-screen inventory

Ballbox originated from that hardware + media thesis, but current product/discovery focus is concentrated on a software wedge around coach workflows, where ATC-linked club/court context may create an advantage generic tools do not have.

## Why it could work

- captive audience in venue
- recurring need for balls
- visible hardware as media surface
- software can connect payments, playback, and reporting
- software may also create a wedge in coach operations if Ballbox can use ATC-connected context better than generic tools

## Still missing

- pricing model by stakeholder
- proof that ad demand exists at useful rates
- proof that clubs want the machine under realistic commercial terms
- proof that operations stay simple enough
- proof that the coaches wedge is both painful enough and ATC-differentiated enough
